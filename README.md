# 🍅 Pomodoro Study Timer

A clean, minimal Pomodoro timer built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

## ✨ Features

- **25-minute focus** sessions and **5-minute break** sessions
- Circular SVG progress ring that counts down in real time
- Audio alarm + pop-up alert at the end of each session
- Auto-switches between Study and Break modes after each alert
- Tracks **Pomodoros completed** and **total focus time** for the day
- Fully responsive — works on desktop and mobile

## 🚀 Getting Started

1. Download or clone the repository
2. Open `index.html` in any browser
3. Hit **Start** and get to work!

No build tools or installation required.

## 🗂 File Structure

```
pomodoro/
├── index.html   # App structure and layout
├── style.css    # Dark theme styling
└── script.js    # Timer logic and alarm flow
```

## 🛠 Built With

- HTML5
- CSS3 (custom properties, SVG animation)
- Vanilla JavaScript (Web Audio API for alarms)
